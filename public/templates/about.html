What is this?

This is a project that I (bjdixon) have been playing with. It's a utility that should allow one to look up bus or streetcar routes and ultimately receive predictions for the next arrival at specified stops. It is in now way associated with the TTC and if you decide to use it you do so entirely at your own risk.

The data is provided by NextBus via a self hosted version of the Restbus JSON API.

Any issues can be logged at the project issues page and pull requests will be gratefully received.
